# Obsidian 图片管理插件 - 小红书发布方案

---

## 📱 发布策略

小红书适合**多图轮播**形式，建议制作 5-9 张图文卡片，每张卡片一个核心卖点。

---

## 🎨 配图规格

| 项目 | 规格 |
|------|------|
| 尺寸 | 1080 x 1440px (3:4 竖图) 或 1080 x 1080px (1:1 方图) |
| 风格 | 简洁科技风 / 笔记工具类 |
| 配色 | 紫罗兰(#7B68EE) + 白色 + 深色文字 |
| 字体 | 标题用粗体，正文用常规体 |
